## How to use

1. Browse the [catalog](#catalog) below.
2. Copy the **Markdown** or **HTML** snippet under any asset.
3. Paste it into your README. Done.

Images are served straight from `raw.githubusercontent.com` — no CDN, no versioning, just a URL:

```markdown
![Active](https://raw.githubusercontent.com/zrosenbauer/art/main/art/banners/banner_active/banner_active.png)
```

Want to add or remix something? See [CONTRIBUTING.md](./CONTRIBUTING.md).

## Skillicons

256×256 skillicons.dev-compatible tool icons — flat single-color foreground on a brand-color squircle (rx=60), centered to ~55–65% canvas occupancy to match skillicons.dev visual weight. When mixing with a `skillicons.dev/icons?i=…` bundle in the same row, add `hspace=4` per `<img>` (or wrap the row in a flex container with `gap: 8px`) — the bundle endpoint bakes a ~8px gap between icons that single-image embeds don't get.
